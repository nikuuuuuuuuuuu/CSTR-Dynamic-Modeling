# CSTR-Dynamic-Modeling
 Open-Loop Dynamic Modeling and Stability Analysis of a Non-Isothermal CSTR
 <img width="1408" height="768" alt="cstr_modelling_parameters" src="https://github.com/user-attachments/assets/6ba4f168-442b-463e-8cda-51aa8617414d" />

This repository contains the mathematical modeling and dynamic simulation of a non-isothermal Continuous Stirred-Tank Reactor (CSTR) developed in **MATLAB & Simulink**. The project analyzes the open-loop stability, thermal equilibrium, and reactant conversion profiles under highly coupled non-linear conditions
# Mathematical Model & Governing Equations
The system is modeled using state-space differential equations derived from fundamental mass and energy balances:
1) Component Balance (Mass Balance)
This equation tracks the rate of change of reactant concentration (C_A) inside the reactor over time:
