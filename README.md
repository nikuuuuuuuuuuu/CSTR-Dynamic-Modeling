# CSTR-Dynamic-Modeling
 Open-Loop Dynamic Modeling and Stability Analysis of a Non-Isothermal CSTR
This repository contains the mathematical modeling and dynamic simulation of a non-isothermal Continuous Stirred-Tank Reactor (CSTR) developed in **MATLAB & Simulink**. The project analyzes the open-loop stability, thermal equilibrium, and reactant conversion profiles under highly coupled non-linear conditions
